Руководство по установке

Системные требования:

ОС: windows 10-11

ПК: программа работает почти на любых пк

Дополнительно: доп установки программ/библиотек не требуется

Установка:

1. Найдите загруженный файл установки.

2. Дважды щелкните по файлу для начала установки.

3. Следуйте инструкциям мастера установки.

Для установки перейдите в наш ТГ бот
